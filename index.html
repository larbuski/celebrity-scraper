Celebrity Scraper

This code includes the axios library in your HTML page using a script tag, and then uses the global axios object to make a GET request to the IMDB page for the celebrity, and parse the response data using cheerio. It then uses the extracted data to update the celebrity-name and celebrity-portrait elements on the page, and adds an event listener to the refresh-button to refresh the page when clicked. You can use this code as a starting point and modify it to fit your specific needs.
